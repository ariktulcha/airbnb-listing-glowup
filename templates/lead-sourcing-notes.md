# Lead Sourcing Notes — Athens / Madrid / Paris

## Goal

Build a qualified lead list for Airbnb Listing Glow-Up without relying on aggressive scraping.

Use manual research first. The bottleneck is not finding Airbnb listings; it is finding **reachable hosts with weak-but-fixable listings**.

## Search workflow per listing

1. Open Airbnb listing manually.
2. Record URL, title, area, rating, reviews, visible price range.
3. Score photo/copy opportunity.
4. Copy distinctive title phrases or property names.
5. Search the title/property name on Google.
6. Check whether the same listing appears on:
   - Booking.com
   - Vrbo
   - direct property website
   - Google Business Profile
   - Instagram/Facebook
   - property manager website
7. Record contact route.
8. Only create preview if contactability is plausible.

## Contactability signals

Strong:
- direct website with contact form,
- property manager brand,
- Instagram with active posts,
- Booking/Vrbo listing with property name,
- Google Business Profile,
- LinkedIn page/person tied to property.

Weak:
- anonymous Airbnb host only,
- no searchable property name,
- no cross-listing,
- generic title shared by many properties.

## Airbnb search filters to start

Use filters manually; exact settings can change by city.

Baseline:
- Entire place
- 1–2 bedrooms
- 2–4 guests
- central neighborhoods
- rating 4.6+
- visible review count 20+
- not luxury/pro-managed looking
- not new/no-review listings

## Disqualification checklist

Do not pursue if:
- rating below 4.6,
- too few reviews unless listing is clearly high-value,
- property itself looks bad,
- photos are already professional,
- host/contact route is impossible,
- price seems too low for $300 service,
- outreach would require messaging through Airbnb as a fake guest.

## First benchmark targets

Per city after first 30 researched listings:

| Metric | Good | Bad |
|---|---:|---:|
| Leads scoring 70+ | 8+ | <4 |
| Contact found | 30%+ | <10% |
| Obvious visual gap | 40%+ | <20% |
| ADR fit | 60%+ | <30% |

If a city has poor contactability, deprioritize it even if listing quality gaps are obvious.
