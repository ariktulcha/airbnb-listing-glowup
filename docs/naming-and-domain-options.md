# Naming and Domain Options

## Recommendation

Use **Listing Glow Studio** as the company/service brand.

Why:
- It avoids building the company name directly on the Airbnb trademark.
- It is clear enough that hosts understand the offer.
- It sounds more premium than “AI listing tool.”
- It can expand later to Vrbo, Booking.com, direct booking pages, and property managers.
- RDAP check showed `ListingGlowStudio.com` as possibly available at the time checked.

## Suggested brand architecture

- **Company / service:** Listing Glow Studio
- **Productized offer:** Full Listing Glow-Up — $300
- **Founder/avatar:** Arik, the Listing Doctor
- **Internal diagnostic persona:** The Listing Doctor
- **Lead magnet:** Airbnb Listing Scorecard

This lets the site still say “Airbnb listing transformation” for search and clarity, while the actual brand is not dependent on Airbnb.
