# Next Tasks — Listing Glow Studio

## Immediate priority: prove the offer, not more design

The site is now good enough to start selling. The next work should create proof and distribution.

## Task 1 — Create 3 real sample transformations

Goal: replace mock proof with real before/after examples.

Actions:
1. Pick 3 weak listings from different property types.
2. Create a watermarked mini-audit for each.
3. Show: current cover, recommended cover, first 5 photo order, title rewrite, 5 top fixes.
4. Save each as a case-study PDF/image.

Acceptance criteria:
- 3 case-study assets ready to show privately or on the site.
- Each has permission-safe screenshots/assets or fully mocked versions.

## Task 2 — Pilot outreach sprint

Goal: validate whether hosts respond before building more product.

Actions:
1. Build a list of 50 potential hosts/property managers.
2. Prioritize those with decent properties but weak presentation.
3. Send a short, non-creepy message offering a free watermarked listing score.
4. Track replies, objections, and bookings.

Acceptance criteria:
- 50 contacts logged.
- 10+ replies or clear learning on channel/contactability.
- At least 3 free audits delivered.

## Task 3 — Replace placeholder testimonials with real proof

Goal: trust.

Actions:
1. After every pilot, ask for a one-line quote.
2. Ask permission to use property type/city, e.g. “2BR apartment, Athens”.
3. Replace “review line to earn” cards on the homepage.

Acceptance criteria:
- 2–3 real proof cards live.

## Task 4 — Create the actual delivery template

Goal: make the $300 product operational.

Actions:
1. Build a paste-ready delivery file template.
2. Include: score, photos, photo order, titles, description, captions, reel plan, pricing/competitor note, top 10 fixes.
3. Create one completed sample delivery.

Acceptance criteria:
- One reusable delivery template.
- One completed sample file.

## Task 5 — Upwork launch package

Goal: get clients from a marketplace where hosts/property managers already ask for help.

Actions:
1. Upwork profile headline.
2. Project Catalog offer: “Airbnb Listing Glow-Up”.
3. 5 proposal templates.
4. Portfolio assets: site, sample audit, scorecard, delivery sample.

Acceptance criteria:
- Upwork profile ready.
- First 20 proposals sent.

## Task 6 — Email follow-up automation

Goal: capture and convert free score leads.

Actions:
1. Connect Resend or another email provider.
2. Send notification to owner when form submits.
3. Send lead magnet / next-step email to host.
4. Create 3-email follow-up sequence.

Acceptance criteria:
- Form submission triggers owner notification and lead follow-up.

## Task 7 — Design v2 after data

Do not redesign again blindly. Redesign based on:
- objections from hosts
- whether they understand the $300 offer
- which sections they mention
- whether samples convert better than the homepage

Acceptance criteria:
- One redesign brief based on real prospect feedback.
