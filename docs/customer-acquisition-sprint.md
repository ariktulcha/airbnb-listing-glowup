# Customer Acquisition Sprint

## Goal

Win the first 3-5 paying Listing Glow Studio clients without waiting for SEO or social proof to compound.

Primary success metric: **qualified conversations with hosts/property managers**.

Secondary metrics:
- 100 qualified prospects researched
- 50 personalized outbound messages sent
- 10 replies
- 5 preview audits delivered
- 3 paid glow-up orders

## Strategic rule

Do not overbuild the brand before we sell. The order is:

1. credible offer
2. visible sample proof
3. qualified lead list
4. personalized outreach
5. marketplace proposals
6. partnerships
7. SEO/social compounding

## ICP priority

### ICP 1 — individual hosts with strong properties but weak listings

Best signals:
- attractive property but weak cover photo
- random first five photos
- generic title
- thin description
- missing captions
- $100+ nightly rate
- listing looks under-merchandised vs nearby competitors

Avoid:
- luxury listings that already look agency-managed
- extremely low nightly rate listings
- hosts with no public contact path
- listings where the property itself is the issue

### ICP 2 — small property managers with 3-30 listings

Best signals:
- inconsistent photo quality across listings
- copy is duplicated or generic
- several properties in one market
- has website, Instagram, LinkedIn, or public email
- likely to buy multiple units if first result feels useful

### ICP 3 — Airbnb photographers and co-hosts

These are partner targets, not direct buyers. They already sell to hosts and can bundle Listing Glow Studio as an add-on.

## 14-day execution plan

### Days 1-2 — conversion foundation

- [ ] Confirm the offer ladder:
  - Free listing score
  - $99-$149 quick audit / lite rewrite

  - $750+ multi-listing manager package

- [ ] Publish the scorecard lead magnet
- [ ] Create LinkedIn company page
- [ ] Create Facebook page

### Days 5-7 — outbound sprint

- [ ] Research first 100 prospects
- [ ] Qualify to 50 strong targets
- [ ] Send 10 personalized messages per day
- [ ] Track every touch in the CRM
- [ ] Create 5 watermarked mini-previews for the best leads

### Days 8-10 — marketplaces

- [ ] Launch Upwork profile using `templates/upwork-profile.md`
- [ ] Launch Fiverr gig using `templates/fiverr-gig.md`

### Days 11-14 — partnerships and iteration

- [ ] Contact 20 Airbnb photographers/co-hosts using `templates/partnership-outreach.md`
- [ ] Follow up with all non-replies
- [ ] Record objections verbatim
- [ ] Update website copy based on real replies
- [ ] Decide whether the first offer should be $99 audit or $300 full package

## Daily CMO operating cadence

### Morning — list building, 60-90 minutes

1. Add 10-20 new candidates.
2. Score them using the CRM columns.
3. Pick the top 10 for outreach.
4. Write one custom observation per prospect.

### Afternoon — outreach, 45-60 minutes

1. Send 10 messages.
2. Reply to all responses.
3. Create previews for hot replies.
4. Send follow-ups to leads from 3-4 days ago.

### Evening — learning loop, 20 minutes

1. Count messages sent / replies / previews / sales.
2. Record the most common objection.
3. Rewrite one line in the offer or outreach script if needed.

## Channel priorities

| Manual outbound | P0 | Fastest path to conversations |
| Upwork | P0 | Hosts/property managers already searching for help |
| Partnerships | P1 | Photographers/co-hosts can refer repeatedly |

## What not to do

- Do not buy fake reviews.
- Do not backdate fake client posts.
- Do not guarantee bookings, ranking, occupancy, or revenue.
- Do not mass-scrape Airbnb, LinkedIn, or Google Maps.
- Do not send generic bulk messages.
- Do not shame hosts publicly.

## Sales narrative

The host is not buying copy. They are buying **guest trust in the first 10 seconds**.

Core sentence:

> Your property may not be the problem. The listing may be underselling it.

Supporting proof:
- cover photo logic
- first five photo sequence
- title clarity
- captions that answer guest objections
- description structure
- competitor context
- paste-ready delivery

## Minimum viable CRM workflow

Stages:
1. Candidate
2. Qualified
3. Contact found
4. Contacted
5. Replied
6. Preview promised
7. Preview sent
8. Offer sent
9. Won
10. Lost
11. Follow up later

Hot lead criteria:
- strong property, weak presentation
- clear contact route
- $100+ nightly rate or multi-listing operator
- one specific observation we can mention
- ethical outreach path
